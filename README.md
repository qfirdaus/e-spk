# e-base

**e-base** ialah projek asas (base system) untuk pembangunan aplikasi dalaman berasaskan PHP,
direka sebagai platform modular, selamat dan mudah dikembangkan untuk kegunaan organisasi.

## 🎯 Objektif
- Menjadi asas (base) kepada projek aplikasi baharu
- Menyediakan struktur konsisten (MVC ringan)
- Mengutamakan keselamatan, kebolehselenggaraan dan prestasi

## 🧱 Ciri Utama
- Authentication & session management
- Role & access control
- Modular controller & page structure
- Multilingual support (MS / EN)
- System configuration & audit logging
- Cache & helper architecture

## 📁 Struktur Ringkas

## ⚙️ Keperluan Sistem
- PHP 8.1+
- Web server (Apache / Nginx)
- Database (MySQL / Sybase – bergantung konfigurasi)

## 🚀 Cara Setup Ringkas
1. Clone repository
2. Konfigurasi database di `app/configuration/`
3. Pastikan folder `app/cache/` boleh ditulis
4. Akses melalui browser

## 🔒 Keselamatan
- Validasi input & output
- Rate limiting helper
- Audit logging
- Struktur JSON selamat
